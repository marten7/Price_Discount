
def：店铺
    1、当前店铺加购商品总价
        1.1、根据【中、英文逗号、空格】将输入字符串切分成【列表】
            a、遍历列表相加价格得到总价
    2、当前店铺加购商品数量总计【n】
        2.1、可根据【n】和价格【列表】得到单价
    3、店铺参与优惠类型
        3.1、无优惠：type：0
            满减：type：1
            打折：type：2
        3.2、店内优惠：type：1
            满减：type：1
            打折：type：2
        3.3、全网优惠：type：2
            满减：type：1
            打折：type：2
def：优惠类型：满减
    1、根据【减号】切分成列表
        1.1、第一项是上限条件
        1.2、第二项是优惠金额
    2、打印原值
    满减：type：1
def：优惠类型：打折
    1、将输入的百分号去掉，换算成【double】
    2、打印原值
    打折：type：2

执行步骤
1、输入商品单价
    计算总价
2、输入优惠
3、判断优惠类型
    根据输入字符串中包含【减号或者百分号】判断执行方法
        满减
        打折
4、判断店铺满足什么优惠类型【type】
    0：无优惠
    1：满减
    2：打折
    01：全网满减
    02：全网打折
    11：满减后满减
    12：满减后打折
    21：打折后满减
    22：打折后打折
5、输出计算之后的总价
6、输出计算之后的单价

python -m http.server --cgi
